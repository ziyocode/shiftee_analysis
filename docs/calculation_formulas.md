# 계산 수식 정리

## 개요
Shiftee 데이터를 분석하여 초과근로 적정성을 판정하는 계산 수식 정리

## 기준 데이터
- **X1 셀**: 해당 월의 마지막 날짜 (예: 11월 → 30일, 12월 → 31일)
- Python 구현: `calendar.monthrange(end_date.year, end_date.month)[1]`

## 계산 필드

### 기본 데이터 (shiftee_data1에서 복사)
| 열 | 헤더명 | 원본 데이터 |
|----|--------|------------|
| D | 소정근로시간 | G열 |
| E | 승인된 근로시간 | H열 |
| F | 실제 근로시간 | J열 |
| K | 유급휴가시간 | L열 |

### 계산 필드

#### G열: 실제 근로시간(결근,퇴근누락포함)
```
G = F + 결근*8 + 퇴근누락*8
```

#### H열: 실제 근로시간(퇴근출근기반) - SUMPRODUCT
```python
# shiftee_data2에서 직원별 출퇴근 시간 합산
for 각 출퇴근 기록:
    근무시간 = (INT(퇴근시간*1440)/1440 - INT(출근시간*1440)/1440) * 24 - 휴게시간*24

H = sum(근무시간) + 결근*8 + 퇴근누락*8
```

#### I열: 표준 근로시간
```
I = 표준근로시간 + 표준최대잔여유급시간
```

#### J열: 표준 근로시간(결근포함)
```
J = 표준근로시간 + 결근*8 + 퇴근누락*8 + 표준최대잔여유급시간
```

#### L열: 법정근로시간
```
L = D - I
```

#### M열: 실제 초과근로시간
```
M = max(0, F - K)
```

#### N열: 실제 초과근로시간(결근,퇴근누락포함)
```
N = max(0, G - L)
```

#### O열: 실제 초과근로시간(조기출근제외) ⭐ 핵심
```python
H_minus_L = H - L
if (H_minus_L > N) or (H > 300):
    O = N
elif H_minus_L < 0:
    O = 0
else:
    O = H_minus_L
```

#### P열: 조기출근 합산
```
P = N - O
```

#### Q열: 법정 초과 근로시간 ⭐
```
Q = X1 / 7 * 10
예: 11월(30일) → 30/7*10 = 42.86시간
    12월(31일) → 31/7*10 = 44.29시간
```

#### R열: 법규 위반(전일까지) ⭐
```
R = X1 / 7 * 12
예: 11월(30일) → 30/7*12 = 51.43시간
    12월(31일) → 31/7*12 = 53.14시간
```

#### S열: 월법규 위반시간
```
S = 12 * 4.3 = 51.6시간
```

#### T열: 월말까지 가능한 초과근로시간
```
T = S - O (단, 음수이면 "가능시간없음")
```

#### U열: 적정성 ⭐ 최종 판정
```
U = "위험" if O >= Q else "정상"
```
**중요**: `>=` 조건 (같거나 크면 위험)

#### V열: 법규 기준초과자
```
V = "법기준초과" if (O != 0 and O >= R) else ""
```

## 검증 결과

### 11월 데이터 (2025-11-01 ~ 2025-11-30)
- **기준일**: 30일
- **Q (법정 초과)**: 42.86시간
- **R (법규 위반)**: 51.43시간
- **위험 직원**: 35명
- **법규초과**: 22명

### 12월 데이터 (2025-12-01 ~ 2025-12-14)
- **기준일**: 31일 (월 마지막 기준)
- **Q (법정 초과)**: 44.29시간
- **R (법규 위반)**: 53.14시간
- **위험 직원**: 0명 (14일치 데이터만 있음)

## 주의사항

1. **X1 셀**: 항상 해당 월의 마지막 날짜를 사용 (end_date.day가 아님!)
2. **적정성 판정**: `O > Q` (크면 위험, **STRICT >**)
3. **법규초과 판정**: `O >= R` (같거나 크면 법기준초과)
4. **H열 계산**: Excel SUMPRODUCT 로직 정확히 구현 필요
   - datetime 객체를 Excel serial date로 변환
   - INT(time*1440)/1440 로직 적용 (분 단위 절삭)
5. **L열 (법정근로시간)**: `D - K` (소정근로시간 - 유급휴가시간)
6. **M열 (실제 초과근로시간)**: `max(0, F - L)` (실제 - 법정)
7. **교대제 직원 제외**: 본직무가 "교대제"인 직원은 분석에서 자동 제외
   - 야간근무 등 특수 근무 패턴으로 인해 일반 직원과 다른 기준 적용 필요
   - 출퇴근 기록(df2)에서도 함께 제외됨
