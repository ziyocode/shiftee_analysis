# Shiftee 초과근로 분석 도구 - 사용 설명서

Windows용 실행 파일 배포판입니다.

## 빠른 시작

### 1. 실행

`ShifteeAnalyzer.exe` 파일을 더블 클릭합니다.

### 2. 로그인 정보 입력

팝업 창에서 다음 정보를 입력합니다:

- **아이디**: Shiftee 로그인 이메일
- **비밀번호**: Shiftee 비밀번호
- **팀 필터** (선택사항): 분석할 팀명
  - 여러 팀은 쉼표로 구분 (예: `뱅킹IS팀,뱅킹통신보안팀`)
  - 비워두면 전체 직원 분석

### 3. 결과 확인

프로그램이 자동으로:
1. ✅ Shiftee에 로그인
2. ✅ 이번 달 데이터 다운로드
3. ✅ 초과근로 분석 수행
4. ✅ Excel 리포트 생성

**결과물:**
- `output/report_YYYYMMDD_HHMMSS.xlsx` - 분석 리포트
- `data/` - 다운로드한 원본 데이터

## 분석 결과 보는 법

### Excel 리포트 색상 코드

- 🟥 **빨간색 행**: 위험 (법정 초과근로 기준 초과)
- 🟧 **주황색 행**: 법규 기준 초과 (52시간 법규 위반)
- ⬜ **흰색 행**: 정상

### 주요 컬럼

| 컬럼 | 의미 |
|------|------|
| **직원** | 직원 이름 |
| **본조직** | 소속 팀 |
| **실제 초과근로시간 (조기출근제외)** | 법정 기준 비교용 초과근로시간 |
| **법정 초과근로시간** | 이번 달 법정 기준 (월별로 다름) |
| **적정성** | "정상" 또는 "위험" |
| **법규 기준초과자** | 52시간 법규 위반 여부 |

## 문제 해결

### Windows Defender 경고

처음 실행 시 Windows Defender가 경고할 수 있습니다.

**해결 방법:**
1. "추가 정보" 클릭
2. "실행" 버튼 클릭

### 로그인 실패

**확인 사항:**
- ✅ 인터넷 연결 확인
- ✅ Shiftee 아이디/비밀번호 확인
- ✅ Shiftee.io 웹사이트 정상 작동 확인

### 팀 필터 사용법

**예시:**

| 입력 | 결과 |
|------|------|
| (비워둠) | 전체 직원 분석 |
| `뱅킹IS팀` | 뱅킹IS팀만 분석 |
| `뱅킹IS팀,뱅킹통신보안팀` | 두 팀만 분석 |

## 자동 제외 대상

다음 직원은 자동으로 분석에서 제외됩니다:
- **교대제 직원**: 야간근무 등 특수 근무 패턴

## 로그인 정보 저장

- ✅ 아이디와 팀 필터는 `shiftee_config.json`에 저장됩니다
- ❌ 비밀번호는 저장되지 않습니다 (보안)
- 저장을 원하지 않으면 로그인 창의 체크박스를 해제하세요

## 기술 지원

문제가 발생하면:
1. `logs/` 폴더의 로그 파일 확인
2. 개발 담당자에게 문의
